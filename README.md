# TimeCapsulor

Minimal V1 frontend bootstrap for a hackathon build.

- Next.js + TypeScript app shell
- Vapi env/config wiring
- basic routes for `/`, `/session`, `/capsules`, and `/capsules/[id]`
- local docs in `docs/timecapsulor/`
