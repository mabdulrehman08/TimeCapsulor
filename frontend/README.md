# Frontend

This folder contains the user-facing TimeCapsulor app.

## Scope

- Next.js + TypeScript
- Vapi Web SDK integration
- browser webcam and microphone capture
- capsule session UI
- upload calls to backend

## Run

```bash
npm install
npm run dev
```

## Important env vars

```env
NEXT_PUBLIC_VAPI_PUBLIC_KEY=
NEXT_PUBLIC_VAPI_ASSISTANT_ID=
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```
